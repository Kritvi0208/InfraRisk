## 📞 SUPPORT GUIDE

**"Dashboard won't start?"**
- Check Python 3.8+: `python --version`
- Check dependencies: `pip install streamlit plotly pandas numpy`
- Run: `streamlit run dashboard_v2_complete.py --logger.level=warning`

**"Charts not rendering?"**
- Clear browser cache: Ctrl+Shift+Delete
- Refresh page: Ctrl+F5
- Check plotly: `pip install --upgrade plotly`

**"Events not working?"**
- Check browser console: F12 → Console
- Verify session state: Streamlit app should show "Connected"

**"Want to modify code?"**
- Edit directly: `dashboard_v2_complete.py`
- Restart: Streamlit auto-reloads on save
- Test changes: Refresh browser
